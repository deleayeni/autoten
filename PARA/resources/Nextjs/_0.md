# Next.js (App Router)

## What this folder covers

The Next.js 15 App Router — the framework that turns your React components into a real website with routing, server rendering, and optimized assets. This is the biggest and most load-bearing folder in Resources.

## Why it matters for this project

AutoTen *is* a Next.js app. Every page, every `<Link>`, every `<Image>`, every `app/*/page.tsx` is a Next.js construct. You cannot separate "the website" from "Next.js" — they are the same thing.
