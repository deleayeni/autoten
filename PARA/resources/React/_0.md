# React

## What this folder covers

The React concepts you used in AutoTen — component model, hooks, and integrating third-party React libraries like Embla Carousel. Not a React tutorial from zero; focused on *your* code.

## Why it matters for this project

Every visible thing on the site is a React component. The interactive bits (scroll-aware navbar, carousel, hamburger menu) are all hook-driven. Without solid React fundamentals, the client components in [src/components/](../../../src/components/) are opaque.

## Planned lectures (reading order)

1. `_1-component-model.md` — function components, props, JSX, `children`, the `key` prop
2. `_2-state-with-usestate.md` — why [NavBar.tsx](../../../src/components/NavBar.tsx) uses `useState` for its `solid` flag
3. `_3-useeffect-and-cleanup.md` — NavBar's IntersectionObserver setup + the `return () => io.disconnect()` cleanup
4. `_4-usecallback-and-dependencies.md` — [HeroCarousel.tsx](../../../src/components/HeroCarousel.tsx)'s `onSelect` pattern
5. `_5-refs-and-imperative-code.md` — Embla gives you a `ref`; what that means
6. `_6-third-party-react-libs.md` — how to read/use libraries like `embla-carousel-react`, the hook-plus-plugin pattern

## Note

Server Components are *not* here — they live in the Nextjs folder because they're a Next.js invention layered on top of React.
