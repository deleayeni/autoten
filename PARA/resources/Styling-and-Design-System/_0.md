# Styling and Design System

## What this folder covers

How AutoTen is styled: Tailwind CSS v4's CSS-first approach, your brand design tokens, typography system, and the shadcn/ui component layer sitting on top.

## Why it matters for this project

Tailwind v4 is *different* from Tailwind v3 — there's no `tailwind.config.js`. Configuration lives inside [globals.css](../../../src/app/globals.css) using `@theme`. If you learned Tailwind before 2024 or read old tutorials, you'll be confused by your own code.

Every visual decision — the red CTAs, the typography, button variants — is driven by tokens and utility classes in this system.
