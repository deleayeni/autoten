# Supabase and Data

## What this folder covers

Everything about talking to the database: Supabase client setup, the PostgREST query builder, Row Level Security, and how to structure data access so it stays clean as the app grows.

## Why it matters for this project

AutoTen's car inventory lives in Supabase. Every vehicle on the site is loaded via [src/data/cars.ts](../../../src/data/cars.ts). The split between server-side and client-side Supabase clients, the environment variables, and the RLS policies are what make this secure and fast.
