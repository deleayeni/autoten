# Tooling and Deploy

## What this folder covers

The build pipeline and deployment story: Node/npm, Turbopack, Git workflow, GitHub Actions CI, and Vercel auto-deploy. Mostly DevOps-adjacent refresher material framed for this specific project.

## Why it matters for this project

You set this up months ago. When something breaks in CI or a deploy fails on Vercel, you need to remember the pieces: what runs where, which branch auto-deploys, what the lint+build check is gating.
