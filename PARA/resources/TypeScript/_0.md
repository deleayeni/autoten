# TypeScript

## What this folder covers

The TypeScript language features you actually used while building AutoTen. Not a full language tour — just what shows up in the code and what you need to read/write it fluently.

## Why it matters for this project

Every `.ts` and `.tsx` file in [src/](../../../src/) runs through the TypeScript compiler. The whole Car data shape, your Supabase query return types, and React component props are all typed. If you don't understand the type annotations, you can't safely change the code.
